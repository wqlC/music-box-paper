music-box-paper
===============

打印您自己的DIY音乐盒纸带。

目前仅适用于15音符的音乐盒。

打印提示
----------------

最好使用大约110磅重的卡片纸。光滑、有光泽的卡片纸效果最佳，但普通的卡片纸也能很好地工作。67磅重的卡片纸也许可以试试看。索引卡可能会不太稳定。

普通打印机或复印纸无法与音乐盒机制配合使用，但是您可以先打印出来作为草稿纸来使用。

务必关闭所有缩放选项，例如“适应纸张大小”，特别是在将信纸尺寸PDF打印到A4或其他尺寸上时。

什么是DIY音乐盒？
-----------------------

这是一个音乐盒装置。不过不是通过弹簧驱动的方式发声，而是当你摇动小手柄时它会播放曲子。而且不像预设好的曲目，你可以自己在纸上打孔告诉它演奏哪些音符以及何时演奏！

这些东西为什么现在才出现？
-----------------------------------------

我不知道。我是通过Vi Hart的一些视频首次了解到这些：

- [折叠时空](https://www.youtube.com/watch?v=WkmPDOq2WfA)
- [莫比乌斯音乐盒](https://www.youtube.com/watch?v=3iMI_uOM_fY)

由于只有15个音符，跨越两个八度，没有升降调，因此它们相对简单。然而一些伟大的艺术作品正是由于其媒介固有的限制而诞生的。

他们也销售额外的音乐盒纸带，就像随套装附赠的那种长条纸带，如果你觉得找卡片纸、打印较短的纸带并将它们粘在一起很麻烦的话。

尺寸
----------

对于15音符音乐盒：

- 宽41毫米

- 沿着纸带分布15个音符，每个间隔2毫米。低C音符中心距离边缘6.5毫米；中间C音符位于中央；高C音符中心距离另一侧边缘6.5毫米。

- 节拍沿纸带每8毫米一个。

- 打孔直径2毫米。
